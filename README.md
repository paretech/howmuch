# showme

Quickly get product properties from your E-commerce website.

## Installation

```sh
python -m venv venv
venv\scripts\activate
pip install git+https://github.com/paretech/showme
```

## Usage example

```sh
venv\scripts\activate
howmuch "men|clearance" -d <url_of_your_site>
```
